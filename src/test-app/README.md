# Test project for using Describo as a Web Component

## Start project
Start project using vite.
```
cd crate-builder-comopnent
npm install
npm run dev:vite
```

## Build project
Build project using vite.
```
cd crate-builder-comopnent
npm install
npm run build:vite
```

## Preview project
```
cd crate-builder-comopnent
npm run preview:vite
```
